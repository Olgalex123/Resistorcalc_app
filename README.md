Программа для подбора необходимого сопротивления из ряда стандартных E12 или E24 в паралель из 2-х или 3-х шт.
Понадобилась для замены подстроечных резисторов на платах DC/DC преобразователей для фиксированных напряжений. 
Монтаж smd резисторов 0805 производится один на другой или "бутерброд" на место подстроечника.
Расчёт можно запустить в любом браузере: сохранить на устройство файл index.html и выполнить его.
В папке rezistorcalc-apk лежат собраные apk файлы предназначеные для работы на андройд устройствах, выбирайте нужную Вам архитектуру аппарата и устанавливайте.
