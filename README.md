Программа для подбора необходимого сопротивления из ряда стандартных E12 или E24 в паралель из 2-х или 3-х шт.
Понадобилась для замены подстроечных резисторов на платах DC/DC преобразователей для фиксированных напряжений. 
В место штатного подстроечника паяем многооборотный резистор номиналом не ниже штатного. Установив необходимое 
напряжение преобразователя нужно измерить сопротивление многооборотника, отпаяв его от платы DC/DC. 
Измереное сопротивление это и есть номинал, который необходимо подобрать в программе. 
Монтаж smd резисторов 0805 производится один на другой или "бутерброд" на место подстроечника.
Расчёт можно запустить в любом браузере: скачав файл index.html и выполнить его.
В папке rezistorcalc-apk лежат собраные apk файлы предназначеные для работы на андройд устройствах, выбирайте нужную Вам архитектуру(arm64 для телефонов) и устанавливайте.
main.py исходник программы, который можно запустить локально, имея установленый python и необходимое для работы программы. Прграмма на питоне работает в грфическом режиме.
